# Sidekick Project Notes

Sidekick is a private, always-available desktop assistant for meetings,
in-person conversations, and focused work sessions.

The current product direction is:

- local-first meeting capture and transcript-backed notes
- automatic meeting queueing from calendar or meeting links
- configurable global hotkeys and movable overlay controls
- model downloads handled inside the app for non-technical users
- one canonical app mark: `assets/icons/sidekick.svg`
